## 📥 收集箱说明

每个学习模块都有 `inbox/` 目录，用于：

- 存放**未整理**的学习笔记
- 记录**临时想法**和碎片化知识点
- 收集**待分类**的学习资料

**使用建议：**
1. 随时记录，不拘格式
2. 定期整理到正式目录
3. 整理完成后清理收集箱
