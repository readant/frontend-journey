# 准备阶段

本目录包含前端开发入门前需要掌握的基础知识和工具配置。

## 目录结构

```
00-preparation/
├── 01-environment-setup/    # 开发环境搭建
│   ├── 01-code-editors/     # VS Code、WebStorm 等编辑器
│   ├── 02-browsers/         # 浏览器及开发者工具
│   ├── 03-version-control/  # Git 和 GitHub
│   ├── 04-package-managers/ # Node.js、npm、yarn、pnpm
│   ├── 05-build-tools/      # Webpack、Vite、Parcel
│   ├── 06-terminals/        # Windows Terminal、PowerShell
│   ├── 07-shortcuts/        # 快捷键速查
│   └── README.md
├── 02-logic-thinking/       # 思维与逻辑训练
│   ├── 01-logic-basics/     # 逻辑基础
│   ├── 02-algorithm-intro/  # 算法入门
│   └── 03-programming-thinking/ # 编程思维
└── 03-common-tools/         # 通用工具
    ├── 01-cli-tools/        # 命令行工具
    ├── 02-dev-helpers/      # 开发辅助工具
    └── 03-productivity/     # 效率工具
```

## 学习目标

### 环境搭建

- 掌握代码编辑器的配置和使用
- 熟练使用浏览器开发者工具
- 掌握 Git 版本控制
- 熟悉 Node.js 和包管理器

### 思维与逻辑

- 培养逻辑思维能力
- 理解基础算法概念
- 建立编程思维模式

### 通用工具

- 掌握常用命令行工具
- 了解开发辅助工具
- 使用效率工具提升开发效率

## 学习路径

1. **第一步**：配置开发环境（代码编辑器、浏览器、Git）
2. **第二步**：学习基础逻辑和编程思维
3. **第三步**：掌握常用工具和快捷键
4. **第四步**：开始学习 HTML/CSS/JavaScript
