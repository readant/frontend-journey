# VS Code 快捷键手册

## 基础操作

| 快捷键 | 功能 |
| :--- | :--- |
| `Ctrl + S` | 保存文件 |
| `Ctrl + Shift + S` | 另存为 |
| `Ctrl + N` | 新建文件 |
| `Ctrl + W` | 关闭当前文件 |
| `Ctrl + Shift + W` | 关闭所有文件 |
| `Ctrl + Tab` | 切换文件 |
| `Ctrl + 1/2/3` | 切换到第1/2/3个编辑器 |

## 自定义快捷键

在 `File > Preferences > Keyboard Shortcuts` 中可以自定义快捷键。

### 常用命令 ID

- `workbench.action.files.save` - 保存
- `workbench.action.files.newFile` - 新建文件
- `editor.action.commentLine` - 注释行
- `editor.action.formatDocument` - 格式化文档
- `workbench.action.terminal.focus` - 聚焦终端

## 效率技巧

1. **命令面板**: `Ctrl + Shift + P` 是最强大的快捷键，可以执行几乎所有操作
2. **快速打开**: `Ctrl + P` 输入文件名快速定位
3. **多光标编辑**: 按住 `Alt` 点击可以创建多个光标
4. **括号匹配**: `Ctrl + Shift + \` 跳转到匹配的括号
5. **折叠代码**: `Ctrl + Shift + [` 折叠，`Ctrl + Shift + ]` 展开
