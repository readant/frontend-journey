## 🎯 通用快捷键

| 快捷键 | 功能 |
| :--- | :--- |
| `Ctrl + T` | 新建标签页 |
| `Ctrl + W` | 关闭当前标签页 |

| `Ctrl + 1-9` | 切换到第1-9个标签页 |
| `Ctrl + Shift + T` | 恢复关闭的标签页 |
| `Ctrl + R` | 刷新页面 |

| `Ctrl + F5` | 强制刷新 |

| `Ctrl + J` | 打开下载记录 |
| `Ctrl + N` | 新建窗口 |
| `Ctrl + Shift + N` | 新建无痕窗口 |

| `Alt + →` | 前进 |
| `F5` | 刷新页面 |
| `F11` | 全屏模式 |
| `Esc` | 退出全屏/关闭弹窗 |

## 💡 使用技巧

1. **快速导航**: `Ctrl + L` 聚焦地址栏后，输入 `history` 快速访问历史记录
2. **关闭多个标签**: 按住 `Ctrl` 点击多个标签，然后按 `Ctrl + W` 一次性关闭
3. **固定标签**: 右键标签选择 "Pin tab"，适合常用网站
4. **静音标签**: 右键标签选择 "Mute tab"，关闭自动播放声音
5. **任务管理器**: `Shift + Esc` 打开浏览器任务管理器，查看内存占用
