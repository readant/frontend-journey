## 学习目标

- 掌握常用代码编辑器的配置和使用
- 熟练使用浏览器开发者工具进行调试
- 掌握 Git 版本控制与 GitHub 协作
- 熟悉 npm/yarn/pnpm 包管理器
- 了解主流构建工具的基本使用

## 学习顺序

1. **第一步**：安装代码编辑器（推荐 VS Code）
2. **第二步**：安装浏览器（推荐 Chrome）并熟悉开发者工具
3. **第三步**：安装 Git 并配置 GitHub
4. **第四步**：安装 Node.js 和包管理器
5. **第五步**：了解构建工具（Webpack/Vite）
