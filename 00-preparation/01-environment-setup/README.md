# 🛠️ Environment Setup

This directory documents the development environment configuration and tool usage guides for frontend development.

## 📁 Directory Structure

```
01-environment-setup/
├── 01-code-editors/          # Mainstream IDE and editor configurations
│   ├── 01-vscode/           # VS Code configuration
│   ├── 02-webstorm/         # WebStorm configuration
│   ├── 03-sublime/          # Sublime Text configuration
│   └── 04-others/           # Other editors
├── 02-browsers/             # Browsers and developer tools
│   ├── 01-chrome/           # Chrome browser
│   ├── 02-firefox/          # Firefox browser
│   ├── 03-edge/             # Edge browser
│   └── 04-devtools/         # Debugging guides
├── 03-version-control/      # Git and code hosting
│   ├── 01-git-setup/        # Git installation and configuration
│   ├── 02-github/           # GitHub usage
│   └── 03-commands/         # Git commands reference
├── 04-package-managers/     # Node.js and package management
│   ├── 01-nodejs/           # Node.js installation
│   ├── 02-npm/              # npm usage guide
│   ├── 03-yarn/             # yarn usage guide
│   └── 04-pnpm/             # pnpm usage guide
├── 05-build-tools/          # Build tools
│   ├── 01-webpack/          # Webpack configuration
│   ├── 02-vite/             # Vite configuration
│   └── 03-parcel/           # Parcel configuration
├── 06-terminals/            # Terminal tools
│   ├── 01-windows-terminal/ # Windows Terminal
│   ├── 02-powershell/       # PowerShell configuration
│   └── 03-zsh/              # Zsh configuration
└── 07-shortcuts/            # Keyboard shortcuts
```

## 🎯 Learning Objectives

- Master configuration and usage of common code editors
- Proficient in using browser developer tools for debugging
- Master Git version control and GitHub collaboration
- Familiar with npm/yarn/pnpm package managers
- Understand basic usage of mainstream build tools
