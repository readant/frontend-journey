# 🛠️ 环境搭建

本目录记录前端开发所需的开发环境配置和工具使用指南。

## 📁 目录结构

```
01-环境搭建/
├── 01-代码编辑器/          # 主流IDE和编辑器配置
│   ├── 01-VScode/          # VS Code配置
│   ├── 02-WebStorm/        # WebStorm配置
│   ├── 03-SublimeText/     # Sublime Text配置
│   └── 04-其他编辑器/       # 其他编辑器
├── 02-浏览器工具/           # 浏览器及开发者工具
│   ├── 01-Chrome/          # Chrome浏览器
│   ├── 02-Firefox/         # Firefox浏览器
│   ├── 03-Edge/            # Edge浏览器
│   └── 04-开发者工具指南/   # 调试技巧
├── 03-版本控制/            # Git与代码托管
│   ├── 01-Git安装配置/      # Git安装配置
│   ├── 02-GitHub/          # GitHub使用
│   └── 03-常用命令速查/     # Git命令速查
├── 04-包管理器/            # Node.js与包管理
│   ├── 01-NodeJS安装/       # Node.js安装
│   ├── 02-npm/             # npm使用指南
│   ├── 03-yarn/            # yarn使用指南
│   └── 04-pnpm/            # pnpm使用指南
├── 05-构建工具/            # 工程化构建工具
│   ├── 01-Webpack/         # Webpack配置
│   ├── 02-Vite/            # Vite配置
│   └── 03-Parcel/          # Parcel配置
├── 06-终端工具/            # 终端工具配置
│   ├── 01-WindowsTerminal/ # Windows终端
│   ├── 02-PowerShell/      # PowerShell配置
│   └── 03-Zsh与OhMyZsh/    # Zsh配置
└── 07-快捷键手册/          # 快捷键汇总
```

## 🎯 学习目标

- 掌握常用代码编辑器的配置和使用
- 熟练使用浏览器开发者工具进行调试
- 掌握Git版本控制和GitHub协作
- 熟悉npm/yarn/pnpm包管理器
- 了解主流构建工具的基本用法
