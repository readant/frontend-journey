# 🧠 思维与逻辑入门

编程不仅是写代码，更是一种思维方式。本目录帮助你建立编程所需的逻辑思维基础。

## 📁 目录结构

```
02-思维与逻辑入门/
├── 01-逻辑基础/          # 逻辑思维训练
├── 02-算法入门/          # 基础算法知识
└── 03-编程思维训练/      # 编程思维培养
```

## 🎯 学习目标

### 逻辑基础
- 理解逻辑运算（与、或、非）
- 掌握条件判断和逻辑推理
- 学习真值表和逻辑表达式

### 算法入门
- 理解算法的基本概念
- 掌握常见算法思想（递归、迭代、分治）
- 学习时间复杂度和空间复杂度

### 编程思维训练
- 培养问题分解能力
- 学习抽象思维和模式识别
- 掌握调试和问题解决技巧

## 📋 学习资源

- **书籍**: 《算法图解》、《编程珠玑》
- **在线课程**: freeCodeCamp算法课程
- **练习平台**: LeetCode、HackerRank
