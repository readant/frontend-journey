HTML列表示例

1. 无序列表 <ul>

用于没有顺序关系的项目，默认显示为圆点

购物清单：

  • 苹果
  • 香蕉
  • 橙子
  • 西瓜

嵌套列表：

  • 水果
    • 苹果
    • 香蕉
  • 蔬菜
    • 白菜
    • 萝卜

2. 有序列表 <ol>

用于有顺序关系的项目，默认显示数字编号

学习步骤：

  1. 学习 HTML 基础
  2. 学习 CSS 样式
  3. 学习 JavaScript 语法
  4. 做综合项目

自定义起始编号（start="10"）：

  1. 第十步
  2. 第十一步
  3. 第十二步

不同编号类型（type="A"）：

  1. 选项A
  2. 选项B
  3. 选项C

指定起始值（value 属性）：

  1. 第一步
  2. 第五步（跳过2-4）
  3. 第六步（自动从5+1开始）

3. 定义列表 <dl>

用于术语和定义，dt=术语，dd=定义

前端技术术语：

HTML
HyperText Markup Language，超文本标记语言，用于构建网页结构
CSS
Cascading Style Sheets，层叠样式表，用于美化网页样式
JavaScript
一种编程语言，用于实现网页的交互功能

4. 菜单列表 <menu>

用于一组交互式命令，语义上表示可执行操作
