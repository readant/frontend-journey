# 逻辑思维训练

编程不只是写代码，更是一种思维方式。本目录帮助你建立编程所需的逻辑思维基础。

## 目录结构

```
02-logic-thinking/
├── 01-logic-basics/          # 逻辑思维训练
├── 02-algorithm-intro/       # 基础算法知识
└── 03-programming-thinking/  # 编程思维培养
```

## 学习目标

### 逻辑基础

- 理解逻辑运算（与、或、非）
- 掌握条件判断和逻辑推理
- 学习真值表和逻辑表达式

### 算法入门

- 理解基础算法概念
- 掌握常见算法思想（递归、迭代、分治）
- 了解时间复杂度和空间复杂度

### 编程思维

- 培养问题分解能力
- 学习抽象思维和模式识别
- 掌握调试和问题解决技巧

## 推荐资源

- **书籍**：《算法图解》、《编程珠玑》
- **在线课程**：freeCodeCamp 算法课程
- **练习平台**：LeetCode、HackerRank
