# 准备阶段

本目录包含前端开发入门前需要掌握的基础知识和工具配置。

## 学习顺序

| 顺序 | 模块 | 内容 |
|:----:|------|------|
| 1 | [环境搭建](/00-preparation/01-environment-setup/) | 编辑器、浏览器、Git、Node.js |
| 2 | [逻辑思维](/00-preparation/02-logic-thinking/) | 逻辑基础、算法入门、编程思维 |
| 3 | [通用工具](/00-preparation/03-common-tools/) | 命令行、开发辅助、效率工具 |

## 目录结构

```
00-preparation/
├── 01-environment-setup/    # 开发环境搭建
├── 02-logic-thinking/       # 思维与逻辑训练
└── 03-common-tools/         # 通用工具
```

## 学习目标

- 掌握代码编辑器的配置和使用
- 熟练使用浏览器开发者工具
- 掌握 Git 版本控制
- 熟悉 Node.js 和包管理器
