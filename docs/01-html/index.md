# HTML 学习模块

系统化的 HTML 学习笔记，从基础到实战。

## 学习顺序

| 顺序 | 模块 | 内容 |
|:----:|------|------|
| 1 | [基础入门](/01-html/01-basics/) | 文档结构、文本标签、列表、链接、图片、表格 |
| 2 | [语义化标签](/01-html/02-advanced/) | header、nav、main、article、section 等语义标签 |
| 3 | [表单元素](/01-html/03-forms/) | 表单结构、input 类型、验证、无障碍 |
| 4 | [多媒体标签](/01-html/04-media/) | 视频音频、Canvas 绘图、HTML5 新标签 |
| 5 | [实战项目](/01-html/05-projects/) | 个人简历 + 简易博客 |

## 目录结构

```
01-html/
├── 01-basics/           # 基础入门
├── 02-advanced/         # 语义化标签
├── 03-forms/            # 表单元素
├── 04-media/            # 多媒体标签
├── 05-projects/         # 实战项目
├── inbox/               # 收集箱（未整理笔记）
└── roadmap.html         # 交互式学习路线图
```

## 学完你将能

- 独立编写结构清晰的 HTML 页面
- 正确使用语义化标签组织页面结构
- 制作包含表单、多媒体的完整网页
- 完成个人简历和博客首页两个实战项目
