CSS 特异性

1. 特异性层级

同一段落应用不同特异性的选择器，观察颜色变化：

标签选择器 p: 灰色 - 特异性 0,0,0,1

类选择器 .class-demo: 绿色 - 特异性 0,0,1,0

ID选择器 #id-demo: 蓝色 - 特异性 0,1,0,0

内联样式: 紫色 - 特异性 1,0,0,0

!important: 橙色 - 最高优先级

2. 组合选择器特异性

组合选择器的特异性是各部分的总和：

组合选择器演示

class="highlight" - 红色（ID+类 优先级更高）

3. 源顺序
