CSS 单位与值

1. 绝对单位

固定尺寸，不会随上下文变化：

px 像素：16px

pt 点：12pt

cm 厘米：0.5cm

2. em 和 rem

相对单位，更适合响应式设计：

em（相对于父元素）

父元素：16px

子元素：1.2em = 19.2px
孙元素：0.8em = 15.36px

rem（相对于根元素）

根元素(html)：16px

子元素：1.2rem = 19.2px
孙元素：0.8rem = 12.8px

3. 视口单位

基于浏览器视口的尺寸：

5vw = 视口宽度的 5%

3vh = 视口高度的 3%

1vw = 视口宽度的 1%，1vh = 视口高度的 1%

4. 百分比

相对于父元素的百分比：

父元素：400px
子元素：50% = 200px

5. 颜色格式

CSS 支持多种颜色表示方法：

red
#ff0000
rgb()
rgba()
hsl()
hsla()

6. CSS 变量

自定义属性让样式更易维护：

使用 --primary-color 变量
使用 --secondary-color 变量

7. calc() 函数

在 CSS 中进行数学计算：

width: calc(100% - 40px)
