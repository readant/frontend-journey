## 笔记格式说明

- **知识点**：写在 HTML 注释或页面内注释中，详细解释原理
- **代码示例**：浏览器中实际渲染，直观展示效果
- **最佳实践**：以注释形式提供实际开发建议
- **常见问题**：标注常见错误和解决方案
