# CSS 学习路径

本目录包含系统化的CSS学习笔记，从基础到进阶。

## 学习顺序

1. **基础入门** (`01-basics/`)
   - 01-css-intro.html - CSS简介与引入方式
   - 02-selectors.html - 选择器详解

2. **核心概念** (`02-core/`)
   - 01-box-model.html - 盒模型详解 ✓
   - 02-specificity.html - 特异性与层叠 ✓
   - 03-inheritance.html - 继承机制 ✓
   - 04-units.html - 单位与值 ✓

3. **现代布局** (`03-layout/`)
   - 01-flexbox.html - Flexbox弹性布局 ✓
   - 02-grid.html - Grid网格布局 ✓
   - 03-position.html - 定位详解 ✓
   - 04-responsive.html - 响应式设计 ✓

4. **视觉效果** (`04-visual/`)
   - 01-colors.html - 颜色与渐变 ✓
   - 02-typography.html - 字体与文本
   - 03-backgrounds.html - 背景与图片
   - 04-shadows.html - 阴影与滤镜
   - 05-borders.html - 边框与圆角

5. **动画与交互** (`05-animation/`)
   - 01-transitions.html - 过渡动画 ✓
   - 02-animations.html - 关键帧动画 ✓
   - 03-transforms.html - 2D/3D变换 ✓

6. **高级特性** (`06-advanced/`)
   - 01-variables.html - CSS变量
   - 02-filters.html - 滤镜与混合模式
   - 03-custom-properties.html - 自定义属性
   - 04-pseudo-classes.html - 伪类与伪元素

7. **性能与优化** (`07-optimization/`)
   - 01-selector-performance.html - 选择器性能
   - 02-repaint-reflow.html - 重排与重绘
   - 03-best-practices.html - 最佳实践

8. **实战案例** (`08-practice/`)
   - 01-navbar.html - 导航栏
   - 02-card-layout.html - 卡片布局
   - 03-form-styling.html - 表单样式
   - 04-gallery.html - 图片画廊

## 笔记格式说明

- **知识点**：写在HTML注释或页面内注释中，详细解释原理
- **代码示例**：浏览器中实际渲染，直观展示效果
- **最佳实践**：以注释形式提供实际开发建议
- **常见问题**：标注常见错误和解决方案
