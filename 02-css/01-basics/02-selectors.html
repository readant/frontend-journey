🎯 CSS 选择器详解

📖 什么是选择器？

选择器用于选中你想要设置样式的 HTML 元素。

1️⃣ 基础选择器

标签选择器（元素选择器）

直接使用 HTML 标签名作为选择器，选中所有该标签。

/* 选中所有 p 标签 */
p {
  color: red;
}

这是一个红色段落（标签选择器效果）

类选择器（Class Selector）

使用 . 加类名，可重复使用。

/* 选中 class="demo-class" 的元素 */
.demo-class {
  background: #3498db;
  color: white;
}
这是一个带类名的元素

ID 选择器

使用 # 加 ID 名，页面中唯一。

/* 选中 id="demo-id" 的元素 */
#demo-id {
  border: 3px solid #2ecc71;
}
这是 ID 选择器效果（绿色边框）

通配符选择器

使用 * 选中所有元素。

* {
  margin: 0;
  padding: 0;
}

2️⃣ 组合选择器

后代选择器（空格）

选中某元素内部的所有指定后代元素。

.ancestor .descendant { color: purple; }

这是祖先元素

这是后代元素（紫色）

子选择器（>）

只选中直接子元素。

.parent > .child { background: yellow; }
直接子元素（黄色背景）
嵌套的子元素（不受影响）

相邻兄弟选择器（+）

选中紧邻的下一个兄弟元素。

.sibling-a + .sibling-b { color: teal; font-size: 20px; }
兄弟 A 兄弟 B（紧邻 A，受影响） 兄弟 C（不紧邻 A，不受影响）

3️⃣ 属性选择器

/* 选中带有 data-type="special" 的元素 */
[data-type="special"] {
  background: linear-gradient(135deg, #667eea, #764ba2);
}
属性选择器效果

4️⃣ 伪类选择器

用于元素的特殊状态。

.hover-demo:hover {
  background: #e74c3c;
  color: white;
}
鼠标悬停试试

常用伪类

伪类 作用
:hover鼠标悬停状态
:active鼠标按下状态
:focus获得焦点状态
:first-child第一个子元素
:last-child最后一个子元素
:nth-child(n)第 n 个子元素

5️⃣ 伪元素选择器

用于创建或选中元素的特定部分。

.pseudo-element::before { content: "🔥 "; }
.pseudo-element::after { content: " ✓"; }

伪元素示例

⚠️ 注意：伪元素必须设置 content 属性才能显示！
