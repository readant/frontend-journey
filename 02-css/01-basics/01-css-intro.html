📖 什么是 CSS？

CSS（Cascading Style Sheets）即层叠样式表，用于控制网页的外观和布局。

💡 理解要点

HTML 负责网页的结构（骨架），CSS 负责网页的样式（皮肤）

📝 CSS 三种引入方式

1. 内联样式（Inline Style）

直接在 HTML 标签的 style 属性中写 CSS：

<!-- 内联样式示例 -->
<p style="color: blue; font-size: 18px;">这是一段蓝色文字</p>

这是一段蓝色文字（内联样式效果）

2. 内部样式表（Internal Style Sheet）

在 HTML 文件的 <head> 中使用 <style> 标签：

<!-- 放在 head 标签内 -->
<style>
  p {
    color: green;
    font-size: 16px;
  }
</style>

3. 外部样式表（External Style Sheet）

将 CSS 写在独立的 .css 文件中，通过 link 标签引入：

<!-- 放在 head 标签内 -->
<link rel="stylesheet" href="styles.css">

📊 三种方式对比

方式 优点 缺点 使用场景
内联样式 简单直接 难以维护、无法复用 临时测试
内部样式表 无需额外文件 只对当前页面有效 单页面应用
外部样式表 可复用、易维护 需要额外 HTTP 请求 推荐方式

✍️ CSS 基本语法

/* 选择器 { 属性: 值; } */

h1 {
  color: red;
  font-size: 24px;
}
🎯 记忆口诀

选择器选中谁，大括号包样式，属性值用冒号，分号来结尾
