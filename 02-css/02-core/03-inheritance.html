CSS 继承

1. 可继承的属性

字体和文本属性会从父元素传递给子元素：

父元素（18px, 深蓝色）
子元素自动继承字体大小和颜色
孙元素继续继承

2. 不可继承的属性

盒模型属性需要单独设置：

父元素：红色背景，20px padding，5px 边框
子元素：需要单独设置背景、padding 和边框

3. inherit 关键字

观察不同继承方式的效果：

父元素：紫色，18px，粗体

默认继承：紫色，18px，粗体

覆盖继承：红色，14px，正常

inherit 关键字：紫色，18px，粗体

initial 关键字：默认样式

坚持学习，每个概念都让你离目标更近一步

4. 实际应用

在容器上设置字体样式，所有子元素自动继承：

在容器上设置字体、大小、行高和颜色，所有子元素自动继承。

这样可以避免为每个元素重复设置这些属性。

需要特殊样式的元素可以单独覆盖：比如这个高亮文字。
